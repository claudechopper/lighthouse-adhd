The honest version

You don't have a "superpower." You have a different operating system — and most of the productivity advice was written for the other one.

For years the ADHD content economy has sold you variations of the same hit: you're not broken, you're a hidden genius. The dopamine feels good. The data doesn't support it. ADHD is correlated with shorter lifespans, higher unemployment, and real comorbidity load. Calling it a superpower has been documented to make people ask for fewer accommodations and burn out harder.

Worse: AI is a near-perfect ADHD trap. Novelty + variable reward + no closure is the gambling-research formula. The same tool that can become your executive-function prosthetic can also become your new doom-scroll. Most ADHD-AI tools don't even acknowledge this. Lighthouse is built around it.

Backed by Russell Barkley's executive function model, William Dodson's interest-based nervous system framework (PINCH), and METR's 2026 task-horizon data. Not by influencer testimonials.
What's in the box

Five commands. One folder. That's the whole product.

Lighthouse runs in your terminal via Claude Code. Five slash commands handle the entire ADHD execution loop, from morning activation to post-interruption recovery. Your data lives locally in plain markdown — yours forever, no SaaS lock-in, no cloud sync, no vendor controlling your second brain.

/morning

Daily activation

Reads your inbox, energy log, and convergence bet. Writes today's ONE priority and three micro-tasks. Ends on the first 15-minute step. Doing requires zero decisions.

/energy ±3 <activity>

Energy ledger

3-second logging surface. Builds the dataset that tells you which activities actually light you up — not which ones you've been told should.

/brain-dump

Triage with bite

Takes a chaotic dump and sorts it into today/week/someday/delete. Picks ONE 15-minute first step. No options. No alternatives. Commitment.

/sync

Evening off-ramp

Eight-line review. What got done, what rolled, tomorrow's draft priority, one reflection question answerable in 30 seconds. Ends with the literal word "Sleep."

/resume

Post-interruption recovery

The killer command. After a kid, a Slack ping, a doom-scroll: "you were doing X, your next step was Y, do this micro-action now." Re-enter without spiraling.

The two-part framework

Find your goals like a generalist. Execute on them like a system.

Most productivity tools assume you already know what you should be doing. Lighthouse assumes you don't — because if you have ADHD and a half-dozen abandoned obsessions, you probably don't. Two artifacts solve that.

1

The Obsession Audit

A one-sitting, 60-minute exercise that surfaces every rabbit hole you've gone deep on in the last decade. Tag them with five symbols. Find the intersection of still-alive + monetizable + naturally-good. That intersection is the lane no one else can compete with you in.

2

The Convergence Bet

Pick ONE intersection of two lanes. Commit to ONE quarter — not forever. Ship one public artifact a week from that intersection. ADHD brains can't do "forever." They can do "until July."

3

The Energy Ledger

Two weeks of data points. Score everything you do from −3 to +3. Run it through Gemini. The pattern that emerges is more honest than any goal-setting workshop ever was. ADHDers run on energy, not discipline.

FAQ

Anticipated objections.

Do I need to be technical to use this?

Yes — at the level of "comfortable installing Claude Code and running terminal commands." If you can clone a GitHub repo and run npm install, you can use Lighthouse. If those are foreign words, the framework still works, but you'd run it with Notion-only and ChatGPT instead of the slash commands.

Why not just use Notion AI or ChatGPT directly?

You can. Many ADHDers do. The thing those tools lack is structural friction against the ADHD-AI dopamine trap — the variable-reward feedback loop that turns "I'll just plan this real quick" into a three-hour spiral. Lighthouse's slash commands have output specs and termination points. They run, they finish, you go do the work.

Is this just a markdown file? It can't be that simple.

It is that simple. ADHD productivity systems fail because they're too complex. Lighthouse is the minimum viable structure to run the actual framework. The complexity lives in the discipline of using it for 90 days without modifying it — which sounds easier than it is.

What if I don't have ADHD but think I have ADHD-shaped problems?

Most of this works for non-ADHD generalists too. Some pieces (the dopamine-trap guards, the interest-based nervous system framing) are sharper for ADHD specifically. The core — convergence bet, energy ledger, /morning command — works for anyone who has too many obsessions and not enough output.

Is there a paid version coming?

Maybe. The Phase 2 plan includes anti-trap features (focus-block timer, customization-paralysis detector, AI-usage meter) that would justify a one-time Gumroad purchase. But the Phase 0/1 starter pack is free forever. If you want updates, watch the GitHub repo or follow the build-in-public posts.

Is this a substitute for medication, therapy, or diagnosis?

No. Lighthouse is a tool. ADHD is a clinical condition with real biological underpinnings. If you're undiagnosed and suffering, the highest-leverage move is finding a clinician — not a productivity system. Lighthouse works alongside treatment, not instead of it.
